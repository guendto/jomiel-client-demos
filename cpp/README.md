# C++14 -example

## Prerequisites

```
apt-get install \
    protobuf-compiler \
    ca-certificates \
    libprotobuf-dev \
    build-essential \
    libzmq3-dev \
    pkgconf \
    cmake \
    git
```

## Building and running

```shell
mkdir build && cd build
cmake .. && make
./demo [args...]
```
